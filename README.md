# ICC-mens-t20-world-cup-2024-records-analytics


# Problem Statement

Cricket fans, analysts, and coaches often struggle to analyze player and team performance from large amounts of match data. Traditional scorecards do not provide deep insights into batting consistency, bowling efficiency, team standings, or match trends. This project aims to build an interactive analytics dashboard that helps users easily explore player statistics, team performance, and tournament insights using data visualization and predictive analytics.

# Key Points / Objectives

1. Develop interactive Power BI dashboards for batting, bowling, and points table analysis.
2. Help cricket fans and coaches quickly evaluate player performance and match statistics.
3. Perform web scraping using BeautifulSoup to collect real-time cricket data from ESPNcricinfo.
4. Use data transformation and DAX measures to generate meaningful insights and comparisons.
5. Identify top-performing players, winning trends, and team qualification patterns in the T20 World Cup 2024.
